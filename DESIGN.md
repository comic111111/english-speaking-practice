# DESIGN.md - 英语口语情景陪练平台设计规范

## 气质与意象
**关键词**：温暖鼓励、清晰专业、成长进步

**场景意象**：
就像一个明亮的现代学习空间——晨光透过落地窗洒在原木书桌上，墙上贴着鼓励性的便利贴，空气中漂浮着咖啡的香气。既有学习的专注感，又不失轻松友好的氛围。用户在这里不是在完成一项枯燥的任务，而是在与一位耐心的AI导师进行愉快的对话练习。

## 视觉策略
- **界面风格**：清新明快的卡片式布局，模块化设计，清晰的视觉层次
- **图形语言**：圆角卡片、柔和阴影、清晰的图标
- **摄影/图像方向**：温暖的场景插图、友好的角色插画

## 配色方案

### 主色系
- **主色（Primary）**：温暖珊瑚橙 `#FF7B54` - 代表鼓励、活力、进步
  - 来源：晨曦的第一缕阳光，给人希望和能量
  - 用于：主按钮、活跃状态、重要强调

- **辅助色（Secondary）**：柔和青绿 `#4ECDC4` - 代表平静、信任、成长
  - 来源：平静的湖面，让人放松和专注
  - 用于：AI消息气泡、成功状态、正向反馈

### 中性色
- **背景色**：温暖米白 `#FFF8F0` - 不刺眼的阅读背景
- **卡片背景**：纯白 `#FFFFFF` - 清晰的内容区域
- **文字主色**：深灰 `#2D3436` - 易读的主体文字
- **文字辅助**：中灰 `#636E72` - 次要信息、说明文字
- **边框分割**：浅灰 `#DFE6E9` - 柔和的分隔线

### 功能色
- **成功/优点**：清新绿 `#00B894` - 正确、进步
- **警告/注意**：活力黄 `#FDCB6E` - 需要注意
- **错误/改进**：柔和红 `#E17055` - 需要改进的地方
- **信息**：温和蓝 `#74B9FF` - 提示信息

## 字体排版

### 字体族
- **中文**：思源黑体、苹方、微软雅黑（系统字体栈）
- **英文**：Inter、SF Pro Display（清晰易读的现代无衬线字体）
- **代码/特殊**：SF Mono、Consolas（等宽字体）

### 字号层级
- **页面标题**：32px / font-bold / 温暖有力
- **模块标题**：24px / font-semibold / 清晰明了
- **卡片标题**：18px / font-medium / 适中突出
- **正文内容**：16px / font-normal / 舒适阅读
- **辅助说明**：14px / font-normal / 次要信息
- **小字提示**：12px / font-normal / 最小层级

### 行高与间距
- 正文行高：1.6（舒适阅读）
- 标题行高：1.3（紧凑有力）
- 段落间距：16px（清晰分割）

## 布局与响应式

### 容器宽度
- 移动端（< 768px）：100vw - 32px（两侧各留 16px）
- 平板（768px - 1024px）：720px
- 桌面（> 1024px）：960px

### 栅格系统
- 12 栏栅格
- 间距：16px（移动端）/ 24px（桌面端）

### 卡片设计
- 圆角：12px
- 阴影：`0 2px 12px rgba(0, 0, 0, 0.08)`（柔和漂浮感）
- 内边距：20px（舒适的内容区域）

## 动效与交互

### 运动性格
- **整体风格**：轻柔、流畅、不打扰
- **时长**：200ms - 300ms（快速但不突兀）
- **缓动曲线**：ease-in-out（自然的进出）

### 关键动效
1. **流式输出**：打字机效果，每个字符 30-50ms 延迟
   - 给用户"AI正在思考并回复"的真实感
   - 模拟真人对话的节奏

2. **语音输入反馈**：波形动画
   - 橙色波形从中心向外扩散
   - 配合语音识别的实时反馈

3. **场景切换**：淡入淡出 + 轻微缩放
   - opacity: 0 → 1
   - scale: 0.95 → 1
   - 过渡时间：300ms

4. **按钮交互**：
   - hover：轻微上浮 + 阴影加深
   - active：轻微下沉
   - disabled：半透明 + 无动效

5. **评分展示**：数字递增动画
   - 从 0 递增到最终分数
   - 时长：800ms
   - 配合进度条填充动画

## 组件规范

### 对话气泡
- **用户消息**：
  - 靠右对齐
  - 背景：主色（珊瑚橙）渐变
  - 文字：白色
  - 圆角：右上 4px, 右下 4px, 左上 16px, 左下 16px

- **AI消息**：
  - 靠左对齐
  - 背景：辅助色（青绿）浅色版
  - 文字：深灰
  - 圆角：右上 16px, 右下 16px, 左上 4px, 左下 4px

### 场景卡片
- 尺寸：宽度自适应，高度 120px
- 内边距：20px
- 选中状态：橙色边框 2px + 轻微阴影提升
- hover：轻微上浮 4px + 阴影加深

### 评价报告卡片
- 四维评分雷达图：青绿色填充
- 优点列表：绿色图标 + 文字
- 改进建议：橙色图标 + 文字
- 学习建议：蓝色图标 + 文字

## 设计禁忌

### 禁止使用
- ❌ 冷冰冰的科技蓝配色（太严肃，不适合学习氛围）
- ❌ 过于鲜艳刺眼的颜色（干扰学习专注度）
- ❌ 复杂的背景图案（分散注意力）
- ❌ 过度动效（影响性能和学习体验）
- ❌ 模糊不清的图标和文字（影响可读性）
- ❌ 过于拥挤的布局（造成视觉压力）

### 避免倾向
- 不要使用纯黑色背景（压抑感）
- 不要使用过多阴影层级（视觉混乱）
- 不要让动效时长超过 400ms（感觉卡顿）
- 不要在主按钮上使用灰色或冷色调（缺乏鼓励感）

## 响应式断点

### 移动端优先（< 768px）
- 导航栏：固定顶部，高度 56px
- 底部栏：固定底部，显示当前场景，高度 60px
- 对话区：占据中间所有可用空间
- 输入区：固定在底部栏上方

### 桌面端（≥ 768px）
- 左侧导航：固定左侧，宽度 240px
- 主内容区：自适应宽度
- 对话区：居中显示，最大宽度 720px
- 底部场景提示：取消，改为顶部导航显示

## 可访问性
- 所有交互元素支持键盘导航
- 颜色对比度符合 WCAG AA 标准
- 图标配有文字说明
- 语音输入提供文字替代方案
